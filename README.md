# SmartUtility

Console ilova yarating. Dastur ishga tushganda foydalanuvchiga asosiy menyu chiqsin va u tanlov asosida turli bo‘limlarga kirsin. Har bir bo‘limdan chiqishda foydalanuvchi asosiy menyuga qaytish yoki dasturni yopish imkoniga ega bo‘lsin.

## **Asosiy kommandalar**
1. **Number Analyzer**
2. **Geometry Zone**
3. **Text Tools**
4. **Exit**


## 1.0 **Number Analyzer**
Number Analyzer bo'limida sonlarning juft toqligi,musbat manfiyligi va kvadrati, kubini topishga yordam beradi

1. Son juft yoki toqligini aniqlash
2. Son musbat, manfiy yoki nol
3. Sonning kvadrati va kubi
4. Exit

### 1.1  Son juft yoki toqligini aniqlash
Ushbu bo'lmda consoledan kiritilgan soning juft yoki toq ekanligini oson bilib olish mumkin
